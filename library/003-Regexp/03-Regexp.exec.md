## `Regexp.exec`

### 描述

在一个指定字符串中执行一个搜索匹配。返回一个数组或 `null`。

### 参数

string，要匹配正则表达式的字符串。

### 返回值

如果匹配成功，`exec()` 方法返回一个数组，并更新正则表达式对象的属性。

返回的数组将完全匹配成功的文本作为第一项，将正则括号里匹配成功的作为数组元素填充到后面。

如果匹配失败，返回 `null`。

### 示例

```js
var str = 'leftTop';

var reg = /^([a-z]*)([A-Z]\w*)*/;

var result = reg.exec(str);

console.log( result );
```

数组的 `input` 属性：用于匹配正则表达式的原始字符串
