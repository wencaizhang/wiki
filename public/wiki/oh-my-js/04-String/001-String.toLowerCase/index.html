String.prototype.toLowerCase()

Catalogue
  1. 1. 描述
  2. 2. 参数
  3. 3. 返回值
  4. 4. 示例

描述

toLowerCase 会将调用该方法的字符串转为小写形式，并返回。

toLowerCase 不会影响字符串本身。

参数

无

返回值

一个新的字符串，转换为小写形式之后的字符串。

示例

Newer
Array.prototype.includes()
Older
String.prototype.toUpperCase()
